📚 Homework Manager

📝 Project Description

The Homework Manager grew out of a simple but frustrating problem: I was tired of asking for the homework in the class chat and getting no answer.

Of course, you could simply write the homework down on paper or enter it into a personal digital calendar. But that would have been too boring—and would have only helped me.

So I wanted to develop a solution that also benefits my classmates. That is how the Homework Manager came about: a program that offers a homework and exam calendar and includes some additional features.

🔄️ Update 1.5

The user interface was adjusted to guarantee better usability, and the "grade calculator" was implemented.

🔄️ Update 1.6.x

I am currently trying to redesign the user interface, but it quickly became clear that the possibilities of a purely Python interface are very limited or far too complicated to achieve a good result. The solution was to design the interface with HTML and CSS while keeping the functionality in Python.

🔄 Update 1.6.1-BETA

After extensive experiments, the website is now ready and works largely reliably, yet I noticed several potential problem points such as the requirement to download a software package and static assets tied to the software.

🔄 Release 1.6.2

The new Homework Manager website is online. No local installation is required anymore, the web interface can be designed much more flexibly, and assets are stored centrally on the server and can be updated dynamically. Changed timetables are now displayed automatically without installing a new version.

🔄️ Update 1.6.3 & 1.6.4

The website received several visual as well as security improvements, including new overlays. Based on user feedback, a new overlay was created that allows an administrator to edit or delete entries.

🔄️ Update 1.7 incl Patch 00x1 & 00x2

Various tweaks to the user interface now make it possible to create and edit entries directly in the calendar.
